# weixiao
WeiXiao SDK for Go   （腾讯微校SDK）

[腾讯微卡官方文档](https://wiki.weixiao.qq.com/api/)

## 结构
包名遵照官方文档中的服务类型设定，包中相应的文件名也与文档保持一致

- third: 应用服务
- school: 主体信息服务
- guard: 门禁服务
